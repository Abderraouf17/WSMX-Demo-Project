# WSMX Smart Demo Project

## Overview
This project simulates the execution of WSMX (Web Service Execution Environment).

## Features
- Service discovery (filtering)
- Intelligent selection (multi-criteria scoring)
- Data mediation (format conversion)
- Execution simulation

## How to Run
1. Open index.html
2. Select destination
3. Click "Search Services"

## Key Logic
- Discovery → filter()
- Selection → scoring + reduce()
- Mediation → format conversion
- Execution → simulated call

## Note
This is a simulation of WSMX behavior for educational purposes.
